Tentang Website Ini

Website ini menyediakan video pembelajaran, materi premium

Solusi kami: menampilkan preview YouTube (thumbnail) di website sebagai preview video kami. Setelah pembelian, admin akan mengirimkan link Google Drive asli melalui WhatsApp. Kamu juga dapat meminta panduan tambahan terkait instalasi file pada perangkatmu.

Catatan teknis tentang gambar: agar gambar tampil di GitHub Pages, pastikan:

  1. File ditempatkan di folder assets/img/ (sesuai path di HTML).
  2. Nama file tanpa spasi (gunakan dash atau underscore, contoh: wardah-alfarah.jpg).
  3. Gunakan format .jpg/.png/.webp dan ukuran wajar (tidak terlalu besar).

FAQ

Kenapa tidak langsung streaming dari Google Drive?

GitHub Pages dan hosting statis tidak cocok untuk streaming file besar; juga Google Drive embed sering diblok. Oleh karena itu file asli akan kami kirim via WhatsApp setelah pembayaran diverifikasi.

Bisa bayar lewat apa saja?

Kami menerima Dana dan ShopeePay. Nomor tertera di panel setiap paket. Hubungi admin via WA jika ingin metode lain.

Kenapa gambar kadang tidak muncul?

Periksa: path file, ekstensi (jpg/png), dan nama file tanpa spasi. Contoh path yang benar: assets/img/about.jpg.

Apakah konten aman dan original?

Semua materi dibuat untuk tujuan pembelajaran. Jika ada materi yang memerlukan sumber tambahan, admin akan melampirkan referensi.
